– Cho phương trình bậc nhất có dạng: ax^2 + bx + c = 0, với a, b, c được nhập từ bàn phím. Tìm giá trị của x1, x2. 1. Nhập vào các hệ số a, b, c cho phương trình bậc hai. 2. Tính delta = b*b-4*a*c – Nếu Delta < 0 thì phương trình vô nghiệm. – Nếu Delta = 0 thì phương trình có nghiệp kép là x1 = x2 = -b / 2 * a. – Nếu Delta > 0 thì phương trình có hai nghiệm phân biệt là x1 = (-b + Math.sqrt(delta)) / (2 * a) , x2 = (-b – Math.sqrt(delta)) / (2 * a). 3.
